# Task Manager Frontend 🎯

## 📌 Description

A modern frontend application built with Next.js and React that consumes a REST API to manage tasks.

---

## 🛠️ Technologies

* Next.js
* React
* TypeScript
* Tailwind CSS

---

## ⚙️ Features

* User authentication
* Dashboard with tasks
* Create, update, delete tasks
* Real-time UI updates
* Secure API communication with JWT

---

## 📦 Installation

```bash
npm install
```

---

## ▶️ Run Project

```bash
npm run dev
```

---

## 🌐 Backend API

Make sure the backend is running at:

```text
http://localhost:4000
```
